# Almanahil School System

A school management system built with Vue.js 3 (frontend) and ASP.NET Core Web API (backend).

## Project Structure

```
Almanahil-School-System/
├── frontend/          # Vue.js 3 application
├── backend/           # ASP.NET Core Web API
├── docs/              # Documentation
└── README.md
```

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
dotnet restore
dotnet run
```
